= 疎行列

== 疎行列とは

ほとんどの要素が零の行列のこと．


例
\[
\begin{bmatrix}
1 & 2 & 3 & 0\\
0 & 0 & 0 & 1\\
2 & 0 & 0 & 2\\
0 & 0 & 0 & 1\\
\end{bmatrix}
\]

実際にプログラムで扱う場合は，この行列をそのままメモリに展開することはなく，例えば次のように扱う．

[source, c]
----
a = [1, 2, 3, 1, 2, 2, 1]; // 要素の値
r = [0, 0, 0, 1, 2, 2, 3]; // 行index
c = [0, 1, 2, 3, 0, 3, 3]; // 列index
----

== 疎行列の良さ

疎行列 $$A$$ とベクトル $$x$$ の積が $$O(Aの非零要素数)$$ で計算できること．

例えば $$y=Ax$$ を計算するときは，

[source, c]
----
for(int i=0;i<a.length;i++){
    y[r[i]] += a[i]*x[c[i]];
}
----

のようにする．
