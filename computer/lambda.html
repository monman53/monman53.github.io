Untyped Lambda Calculus Programming

In this page, we use De Bruijn index
[https://en.wikipedia.org/wiki/De_Bruijn_index]
 in lambda term, and use Church encoding
[https://en.wikipedia.org/wiki/Church_encoding#Church_numerals]
.

Definition of Lambda Term

<term> ::= n
<term> ::= (λ <term>)
<term> ::= (<term> <term>)

n is natural number.

Numbers, Booleans

In this page, [cn] means natural number n in church encoding.

Natural Numbers
[c0] = (λ(λ1))
[c1] = (λ(λ(2 1)))
[c2] = (λ(λ(2(2 1))))
[c3] = (λ(λ(2(2(2 1)))))
[c4] = (λ(λ(2(2(2(2 1))))))
Booleans
TRUE  = (λ(λ2))
FALSE = (λ(λ1))
      = [c0]
